# Tic Tac Toe in C

## Description
This is a simple Tic Tac Toe game implemented using C programming language. It allows two players to play the game in the opposite.

## Features
- Two player mode
- Input validation
- Win detection
- Draw detection
- Simple console interface
- Rematch challenge

## Technologies Used
- C programming language

## How to Run
1. Compile the program:
   gcc tic_tac_toe.c -o tic_tac_toe

2. Run the program:
     tic_tac_toe.exe
